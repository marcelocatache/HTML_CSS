CSS, HTML E JAVASCRIPT



CSS

Cascading Style Sheets é um mecanismo para adicionar estilo a um documento
web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido
dentro das tags style. Também é possível, em vez de colocar a formatação


HTML

HTML abreviação para a expressão inglesa HyperText Markup Language, que
significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação
utilizada na construção de páginas na Web. Documentos HTML podem ser
interpretados por navegadores. A tecnologia é fruto da junção entre os
padrões HyTime e SGML.


JAVASCRIPT

JavaScript é uma linguagem de programação interpretada estruturada, de
script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente
com HTML e CSS, o JavaScript é uma das três principais tecnologias da
World Wide Web.
